# Parameter Ranking

Для MVP используется экспертный ранг параметров.

## Формула

```text
rank_score = 0.35 * mechanical_significance
           + 0.25 * sensitivity
           + 0.20 * observability
           + 0.20 * decision_value
```

## Приоритеты

- `P0` — критически обязательные поля для инженерной пригодности.
- `P1` — важные поля для уверенной интерпретации и идентификации.
- `P2` — желательные расширяющие данные.
- `P3` — дополнительные каналы и медиа.

## Применение

- индекс информационной обеспеченности;
- identification readiness;
- сортировка критически недостающих данных.
