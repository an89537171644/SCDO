# Import Templates

Шаблоны для типизированного импорта инженерных наблюдений.

Общие колонки:

- `timestamp`
- `value`
- `unit`
- `spatial_location`
- `source_type`
- `quality_flag`

Доступные шаблоны:

- `deflection.csv`
- `displacement.csv`
- `strain.csv`
- `crack_width.csv`
- `settlement.csv`
- `tilt.csv`
- `frequency.csv`
- `acceleration.csv`
- `temperature.csv`
- `humidity.csv`
