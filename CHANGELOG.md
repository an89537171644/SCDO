# Changelog

## v1.1 - Mechanics-ready data layer

- расширена доменная модель элементов новыми механическими и материалоспецифичными полями;
- расширена модель дефекта инженерными полями тяжести, локализации и механизма;
- расширены каналы и измерения направлением оси, знаком, сценарием нагрузки, компенсацией, агрегацией, прибором и калибровкой;
- `observation_package` дополнен блоками `section_properties` и `critical_missing_data_by_element`;
- `information_sufficiency_index` переведен на покрытийно-взвешенную логику;
- `identification_readiness_report` показывает отдельную готовность по механическим классам;
- Streamlit UI дополнен шаблонами элементов, медиа-вкладкой и таблицей заполненности по важным элементам;
- demo dataset и тесты обновлены под новый контракт.
